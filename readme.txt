A simple plugin that allows you to hide the content (or part) of posts and pages to unregistered users.

== Description ==

This plugin lets you hide the content of posts and pages to unregistered users.

The plugin adds a metabox to the edit screen for pages and post and a button to the editor. 

The metabox lets you hide the content of pages and posts individually. The editor button lets you hide part of content of pages and posts.

== Installation ==

1. Unzip the file 'jgc-content-for-registered-users.zip'
2. Upload the 'jgc-content-for-registered-users' folder to the '/wp-content/plugins/' directory

3. Activate the plugin through the 'Plugins' menu in WordPress

4. Go to Dashboard > Settings > JGC Content for Registered Users
